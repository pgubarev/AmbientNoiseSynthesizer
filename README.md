# Ambient Noise Synthesizer

Проект содержит код синтезатора, в основе которого лежит усиление определенных частот для получения ноты из любого звука.
Синтезатор позволяет изменять параметры эквалайзера который используется для училения частоты и подмешивать ноты сгенерированные осцилятором.

## Дорожная карта

[x] Базовая генерация звука на основе загруженного семпла
[x] Управление эквалайзером для корректировки синтезирования нот 
[x] Управление параметрами зацикливания звука
[x] Возможность подмешивать дополнительные ноты к основной генерируемой ноте
[x] Возможность подмешивать звук, генерируемый осцилятором
[x] Возможность настраивать высоту ноты и тип осцилятора
[ ] Добавить lowpass/highpass фильтры для осцилятора
[ ] Корректная логика смешивания для предотвращения перегрузок
